# Sagittal Band: стабілізатор сухожилля розгинача над MCP і синдром боксера

Sagittal band — це поперечна ретикулярна зв'язка на рівні MCP суглоба, яка утримує сухожилля extensor digitorum communis (EDC) точно по центру над п'ясно-фаланговим суглобом. Без неї сухожилля розгинача зміщується убік при згинанні пальця — і пальцем стає неможливо повністю розігнути. Це невелика структура, про яку більшість атлетів не знають до першого прямого удару по MCP суглобу. Після удару — знають добре. Sagittal band rupture, відомий як «boxer's knuckle» або Sagittal Band Syndrome, є найпоширенішою травмою extensor hood і прямим наслідком відсутності розуміння анатомії кулака.

---

## Анатомія: частина extensor hood

Sagittal band є компонентом extensor hood — складного апоневротичного комплексу, що розподіляє силу від EDC і intrinsic muscles (lumbricals, interossei) на фаланги пальця.

**Локалізація:** Рівень MCP суглоба — дорсально.

**Початок:**
- Volar plate (долонна пластина MCP суглоба)
- Transverse metacarpal ligament (поперечна п'ясна зв'язка)

**Прикріплення:**
Sagittal band охоплює сухожилля EDC дорсально з обох боків — радіального і ульнарного — і з'єднується з протилежного боку. Таким чином він формує своєрідну «петлю» або «сідло» навколо EDC над суглобом.

**Обидві стрічки:**
- Radial sagittal band — з радіального боку
- Ulnar sagittal band — з ульнарного боку
- Radial sagittal band травмується значно частіше через механіку прямого удару кулаком

**Іннервація і кровопостачання:** Структура пасивна — зв'язкова. Кровопостачання — з регіональної дорсальної мережі.

**Функції sagittal band:**

**Центрування EDC над MCP:** При згинанні та розгинанні пальця sagittal band утримує сухожилля EDC точно по середній осі MCP суглоба. Без цього фіксатора сухожилля зміщувалося б убік при згинанні під навантаженням.

**Запобігання subluxation:** Якщо sagittal band пошкоджений — EDC зміщується (subluxates) у бік меншого опору. При розриві radial sagittal band сухожилля іде ульнарно.

**Передача сили від intrinsic muscles:** Sagittal band є частиною ланцюга, через який lumbricales і interossei передають силу в extensor mechanism. Без нього ефективність цієї передачі знижується.

---

## Extensor Hood: контекст

Sagittal band — лише один компонент extensor hood (extensor expansion). Повний список:

- **Central slip:** Прикріплення EDC до основи середньої фаланги → розгинання PIP
- **Lateral bands:** Йдуть до дистальної фаланги → розгинання DIP
- **Triangular ligament:** З'єднує lateral bands дорсально на рівні середньої фаланги
- **Transverse retinacular ligament:** Утримує lateral bands від дорсального зміщення
- **Oblique retinacular ligament (Landsmeer):** Координує розгинання PIP і DIP
- **Sagittal band:** Центрує EDC на рівні MCP

Весь цей комплекс перетворює силу EDC + intrinsics у координоване розгинання всіх суглобів пальця. Пошкодження будь-якого компонента порушує механіку всього ланцюга.

---

## Механізм травми: Sagittal Band Syndrome

**Класичний механізм:** Прямий удар по MCP суглобу в положенні згинання кулака — наприклад, closed-fist punch, удар по жорсткій поверхні або прийом важкого м'яча кулаком. Radial sagittal band розривається від раптового зсуву EDC при форсованому згинанні MCP під навантаженням.

**Що відбувається при розриві radial sagittal band:**
- EDC втрачає ульнарну підтримку з радіального боку
- При згинанні MCP сухожилля зміщується в ульнарний бік (до мізинця) — subluxates
- При розгинанні MCP сухожилля може повертатися назад — звідси «клацання» або «щелчок»

**Клінічна картина:**
- Біль над MCP суглобом дорсально
- Відчуття «клацання» або «защіпання» при згинанні і розгинанні пальця
- Труднощі з розгинанням пальця із повністю зігнутого положення
- Можлива деформація при активному розгинанні — EDC видно зміщеним убік

**Який палець страждає найчастіше:**
Long finger (середній, 3-й) MCP — через анатомічне розташування і найбільший ризик прямого контакту при ударі кулаком. Проте index і ring finger також уражаються.

**Ступені розриву:**
- Grade I — розтягнення без повного розриву, EDC залишається центрованим
- Grade II — частковий розрив, intermittent subluxation
- Grade III — повний розрив, постійна subluxation EDC при згинанні

## Клінічна діагностика

**Пальпація дорсуму MCP:** Болючість по ходу EDC над MCP суглобом при пальпації — базовий тест. При subluxation можна пропальпувати зміщення сухожилля.

**Динамічна оцінка:** Попросити пацієнта повільно згинати і розгинати палець — при Grade II–III subluxation EDC видно зміщення убік при згинанні і повернення при розгинанні. «Snap» або «click» при цьому — сильна ознака.

**Elson test:** Використовується переважно для оцінки central slip, але допомагає оцінити загальну цілісність extensor mechanism.

**МРТ:** Золотий стандарт підтвердження. T2-зважені зображення покажуть набряк і розрив band. Важливо для диференціювання Grade II і Grade III перед вибором тактики.

**Консервативне vs хірургічне лікування:**
- Grade I–II (часткові розриви): 6–8 тижнів шинування в положенні розгинання MCP (Extension Splint for MCP). Палець іммобілізований при MCP 0° — EDC залишається центрованим, band загоюється.
- Grade III (повний розрив, хронічна subluxation): Хірургічне відновлення або реконструкція sagittal band. Після хронізації (>3–6 місяців) консервативне лікування малоефективне.

## Тренувальний контекст

**Хто в групі ризику:**

**Ударні єдиноборства** (бокс, MMA, кікбоксинг): Найвищий ризик. Прямий удар по цілі в положенні closed fist з розгинанням MCP у момент імпакту — класичний механізм. Тренування на важких мішках без достатнього тейпування MCP суглобів є фактором ризику.

**Важка атлетика (catching barbell):** Catch у clean або catch у clean and jerk вимагає швидкого переходу від grip до front rack. При падінні грифа на кулак у нестандартній позиції є ризик прямого удару по MCP.

**Гімнастика і кросфіт:** Падіння на кулак (knuckle push-up position) або удар об штангу. Підібрати штангу зі статичного положення при неправильному grip.

**Тімболи і командні ігри:** Прийом твердого м'яча кулаком або відбивання — непоодинокий механізм у тих, хто поєднує силовий тренінг і ігрові види спорту.

**Чому grip training і MCP health пов'язані:**

Сильний grip і розвинені intrinsic muscles не захищають sagittal band від прямого удару — band є пасивною структурою. Проте сильні intrinsic muscles (interossei, lumbricals) краще контролюють MCP у нейтральному положенні при захисних реакціях. Тренування MCP stability через dumbbell wrist rolls, rice bucket і finger extension з resistance band підтримує загальне здоров'я extensor mechanism.

**Повернення до тренінгу після травми:**

- Grade I–II консервативно: Повернення до ударних навантажень не раніше 8 тижнів. Протягом перших 4 тижнів — лише grip-тренінг без ударів і кидків.
- Після хірургії: 3–4 місяці до повного повернення до ударних тренувань. Поступова прогресія від ізометричного grip до dynamic grip і лише потім до striking.

**Протективне тейпування MCP:**

Buddy taping або жорстке MCP taping перед striking sessions є стандартною практикою в бойових мистецтвах. Для силових атлетів, що повертаються після травми або мають рекурентні симптоми — тейпування MCP у легкому розгинанні (15–20°) зменшує ризик повторного subluxation під час grip-heavy тренінгу.

## Підсумок

Sagittal band — поперечна ретикулярна зв'язка на рівні MCP суглоба, що починається від volar plate і transverse metacarpal ligament і охоплює EDC дорсально з обох боків, центруючи сухожилля над суглобом під час руху. Radial sagittal band травмується частіше; його повний розрив призводить до ульнарної subluxation EDC при згинанні — «boxer's knuckle». Середній палець (3rd MCP) уражається найчастіше. Діагностика: клінічна (пальпація, динамічна оцінка) + МРТ для підтвердження ступеня. Лікування: 6–8 тижнів extension splinting при Grade I–II або хірургія при Grade III і хронічних випадках. Силові атлети, що займаються ударними єдиноборствами або catching-heavy дисциплінами, мають найвищий ризик. Sagittal band — пасивна структура, яка не тренується, але може бути захищена тейпуванням і технічно правильним кулаком.
