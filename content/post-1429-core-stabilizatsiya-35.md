# Core і стабілізація для 35+: за межами планки і кранчів

«Тренувати кор» — звучить як окрема активність, щось на кшталт «розминки для спини». На практиці більшість атлетів або ігнорують цілеспрямовану core роботу (вважаючи, що «базові вправи вже навантажують кор»), або виконують щось середнє між планкою і сотнею кранчів наприкінці тренування.

Жоден з цих підходів не відповідає тому, що потрібно атлету 35+.

Перший — частково правильний: базові вправи дійсно навантажують core. Але вони навантажують його як стабілізатор у специфічному руховому контексті — не у всіх напрямках і не з достатньою ізоляцією для виявлення дефіцитів.

Другий — переоцінює кранчі (fleksія хребта) як основний стимул, хоча для більшості атлетів саме здатність опиратись флексії, екстензії і ротації є більш функціонально критичною, ніж здатність виконувати велику кількість кранчів.

Для атлета 35+ правильна рамка для core — це не «скільки планки ти можеш», а «як добре твій trunk може стабілізувати хребет під навантаженням у будь-якому напрямку».

## Що таке core: функціональне визначення

Core — це не «м'язи живота». Це комплекс м'язів, що стабілізують хребет і таз, дозволяючи кінцівкам виробляти і передавати силу без втрати нейтрального положення хребта.

До core функціонально належать: поперечний м'яз живота (transversus abdominis), мультифідус, квадратус лумборум, тазове дно, косі м'язи живота (internal і external obliques), прямий м'яз живота (rectus abdominis), і ряд менших стабілізаторів.

Для атлета 35+ core виконує чотири критичні функції у силовому тренінгу:

**Стабілізація хребта під осьовим навантаженням**: при становленні, присяданні, жимі стоячи. Без достатньої жорсткості trunk хребет отримує надлишковий флексійний або екстензійний момент.

**Передача сили між верхнім і нижнім поясом**: при rowing, жимі, підтягуванні. Силовий «ланцюг» розривається на слабкому core.

**Антиротаційна стабілізація**: при унілатеральних рухах, кабельних вправах, у реальному житті при перенесенні несиметричного навантаження.

**Підтримка внутрішньочеревного тиску (IAP)**: при важких підйомах — правильна техніка дихання і IAP є частиною core стабілізації.

## Чотири напрямки core роботи для 35+

Ефективна core програма для атлета 35+ покриває чотири напрямки навантаження — кожен тренує різну здатність core.

### 1. Антифлексія (Antiflexion)

Опір флексії хребта під навантаженням. Це те, що core робить при становленні і присяданні: не дозволяє хребту округлятись при навантаженні.

**Ключові вправи**: Pallof press (стабілізація у фронтальній площині), dead bug, RKC plank (ізометрична жорсткість), кабельний pull-through в нейтральному положенні.

**Dead bug** — особливо корисний для 35+: виконується лежачи, без осьового навантаження на хребет, ізолює trunk стабілізацію з незалежними рухами кінцівок. Дозволяє відпрацювати нейром'язовий контроль без вартості ціни важкого підйому.

### 2. Антиекстензія (Antiextension)

Опір гіперекстензії поперека. Це те, що core робить при OHP: не дозволяє попереку компенсувати обмежену рухливість грудного відділу.

**Ключові вправи**: rollout (ab wheel або штанга), плавання на суші (prone cobra), hollow body hold.

**Ab wheel rollout** — одна з найбільш недооцінених вправ для core. При правильному виконанні (нейтральний хребет через весь діапазон, без гіперекстензії поперека) — дає максимальний antiextension стимул без ударного навантаження.

### 3. Антиротація (Antirotation)

Опір ротації trunk. Це те, що core робить при single-arm row, кабельних рухах, і у будь-яких унілатеральних рухах.

**Ключові вправи**: Pallof press (антиротаційна варіація), suitcase carry, single-arm farmer carry, half-kneeling cable rotation hold.

**Suitcase carry** — функціонально один з найбільш прямих перенесень у реальне життя: несете важкий предмет в одній руці і підтримуєте нейтральне положення корпусу. Для атлета 35+ — добре навантажує core через нетрадиційний для залу стимул.

### 4. Ротаційна сила (Rotational power)

На відміну від трьох попередніх — це здатність виробляти ротаційний рух, а не опиратись йому. Важливо для спортивної функції і для загального нейром'язового профілю.

**Ключові вправи**: cable woodchop, cable anti-chop, med ball rotational throw.

Для атлета 35+ без спортивних цілей — ротаційна сила є менш пріоритетною, ніж антиротаційна стабілізація. Але повністю виключати її не варто.

## Внутрішньочеревний тиск: що треба знати

Внутрішньочеревний тиск (IAP) — природна система стабілізації хребта при важких підйомах. Вальсальва і правильне дихання у важких підходах є частиною core стабілізації, а не окремим питанням.

Для важких підходів (>80% 1RM) у compound рухах:

- Зробіть глибокий вдих у нейтральній позиції перед підйомом
- Затримайте дихання і «заблокуйте» trunk (Valsalva) на початку руху
- Видихайте у верхній точці або при проходженні мертвої точки

Для помірних ваг (60–80% 1RM):

- Видих при концентрику — природній ритм без форсованої затримки

Для атлета 35+ з серцево-судинними обмеженнями — Valsalva при дуже важких підйомах потребує обговорення з лікарем. Але для більшості здорових атлетів 35+ — правильне використання IAP є частиною ефективного і безпечного важкого підйому.

## Інтеграція core роботи в програму

Core робота для 35+ — це не «10 хвилин в кінці тренування, якщо залишиться час». Це структурна частина тренування.

**Рекомендована інтеграція**:

Як розминка (5–8 хв): dead bug 3×6 кожна сторона, hollow body hold 3×20 сек, bird dog 3×6 кожна сторона. Активує core перед базовою роботою і «вмикає» правильний нейром'язовий паттерн.

Між підходами базових вправ (паузи відновлення): Pallof press або suitcase carry — замість пасивного відпочинку. Не підвищує втому суттєво, але додає антиротаційний і антифлексійний об'єм.

Наприкінці тренування (5–10 хв): ab wheel rollout 3×8–10, cable woodchop або anti-chop 2–3×12.

**Частота**: core 3–4 рази на тиждень. При такій інтеграції — це відбувається автоматично.

## Кранчі і sit-up: яка роль

Прямий м'яз живота (rectus abdominis) тренується кранчами і sit-up. Це реальна вправа з реальним стимулом — але для атлета 35+ без специфічної мети (естетика з фокусом на рельєфі живота) вона є менш пріоритетною, ніж антифлексійна і антиротаційна робота.

Кранч не відповідає функціональній ролі core у силовому тренінгу — де core стабілізує, а не виробляє флексію. Для атлета 35+ з поперековими дискомфортами — кранч може бути виключений повністю без втрати функціональної core сили.

Це не значить, що кранч «шкідливий» — це значить, що він не є найефективнішим вибором для функціональних пріоритетів core 35+.

## Підсумок

Core для 35+ — це стабілізаційна система, а не «вправи для живота». Чотири напрямки: антифлексія, антиекстензія, антиротація, ротаційна сила — охоплюють увесь функціональний спектр. Пріоритет для атлета 35+: антифлексія (dead bug, RKC plank) і антиротація (Pallof press, suitcase carry).

Інтеграція: core як частина розминки і між підходами базових вправ — ефективніше, ніж окремий core-блок наприкінці.
