# Синтез кластеру 2: протокол підготовки нижньої частини тіла

Дев'ять попередніх постів розібрали механізми, вправи, специфіку рухів і типові помилки. Цей пост — операційна карта: що, коли і скільки.

## Дерево рішень: день присідань

```
Тип тренування: присідання

→ Час сесії ≥ 90 хв?
   ТАК → Повний протокол (27 хв):
          General (7) → Ankle mob (3) → Hip CARs+90/90 (5–6) → 
          Glute activation (5) → IAP Dead Bug (3–4) → 
          Рухова репетиція (3) → Навантажені підходи (8–10)
   НІ → Час сесії ≥ 60 хв?
         ТАК → Скорочений (18 хв):
                General (5) → Ankle mob (2) → Hip CARs (4) → 
                Activation 1 серія (3) → Репетиція (2) → Навантажені (7)
         НІ → Компресований (12 хв):
               General (3) → Ankle mob (1×10) → Hip CARs (3) → 
               Activation 1×12 (2) → Навантажені підходи (не скорочувати)

→ Є обмеження дорсифлексії?
   ТАК → Banded mob + heel elevation під час навантажених підходів
   НІ → Стандарт

→ Є обмеження кульшового суглоба?
   ТАК → Banded hip distraction +3 хв + адаптація стійки
```

## Дерево рішень: день становового підйому

```
Тип тренування: становий підйом

→ Варіант: класичний чи sumo?
   Класичний → Стандартний протокол (25 хв):
                General (5–7) → Hip hinge mob: CARs+RDL (5–6) → 
                Thoracic ext (3) → Lats: Y/T+band pull-apart (3) → 
                Glute activation (4) → IAP (3) → Навантажені підходи (8–10)
   Sumo → + Adductor opener + banded distraction в sumo-стійці (+4–5 хв)

→ Час обмежений?
   Скоротити General (5→3 хв) і Thoracic (3→1 хв)
   НЕ скорочувати: hip hinge mob, навантажені підходи
```

## Victor і кластер 2: ключові точки

**RPE першого робочого підходу.** Якщо при стабільному навантаженні RPE першого підходу 1–2 бали вище базелайну — Victor може запитати про структуру розминки.

**Час між початком сесії і першим навантаженим підходом.** Стабільно < 12 хвилин — ознака пропуску мобілізації або активації.

**Асиметрія RPE між сторонами.** При однобічних вправах (сплит-присідання, болгарський присід) — різниця RPE між ногами понад 1 бал є тригером для запиту.

## Що атлет виносить з кластеру 2

- Нижній кінетичний ланцюг має п'ять рівнів; кожен може стати обмеженням.
- Дорсифлексія — найчастіший «прихований» обмежувач присідання.
- Протокол для присідання і для становового підйому принципово різні.
- Активація сідниць — нейронна фасилітація, не тренування. RPE 4–6, не до відмови.
- IAP-праймінг існує і має механізм — не ритуал.
- Постійні анатомічні обмеження не «виправляються» — під них підбирається варіант руху.
- П'ять типових помилок мають спільний знаменник: відсутність розуміння функції кроку.

## Що розглядатиме кластер 3

Кластер 3 (пости 1701–1710) — рухова підготовка верхньої частини тіла для жиму лежачи та жиму над головою. Специфіка: мобілізація плечового пояса, активація ротаторної манжети, грудний відділ під вертикальний жим.
