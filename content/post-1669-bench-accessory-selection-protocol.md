# Протокол добору допоміжних вправ для жиму лежачи: від симптому до вправи

Попередні вісім постів описували окремі обмежувачі жиму лежачи. Цей пост зводить їх у практичний протокол: що спочатку зробити, як поставити діагноз і які вправи вибрати.

---

## Крок 1: Визначити симптом

Запис і відео — обов'язково. Без відео діагностика приблизна.

**Де штанга сповільнюється або зупиняється?**

| Зона | Симптом на відео |
|---|---|
| Нижня точка (0–5 см після торкання) | Немає вибуху, рух «ліпиться» від грудей |
| Середня фаза (90° ліктів) | Сповільнення або зупинка в середині ходу |
| Локаут (остання третина) | Майже дотиск, але не закривається |
| Весь підйом (нестабільно) | Гриф «гуляє», відхиляється від траєкторії |

**Що відбувається з технікою в момент сповільнення?**

| Технічний симптом | Тип дефіциту |
|---|---|
| Лопатки «розходяться» | Позиційний: верхня спина |
| Arch «сідає» | Стабілізаційний: корос або нестача мобільності |
| Лікті розходяться в боки | Позиційний: техніка або слабкий трицепс |
| Гриф відходить від траєкторії | Стабілізаційний: верхня спина або плечо |
| Таз відривається від лавки | Компенсація: leg drive неправильний |

---

## Крок 2: Визначити тип дефіциту

Три типи — як у всьому блоці:

**М'язовий:** м'яз слабкий у конкретній позиції.
- Трицепс → локаут не закривається при правильній техніці.
- Грудні → нижня точка «без вибуху» при стабільному arch.

**Позиційний:** м'яз є, але позиція «ламається» під навантаженням.
- Верхня спина → лопатки розходяться при важкому підйомі.
- Техніка хвату → зап'ясток ламається під навантаженням.

**Стабілізаційний:** система жорсткості не підтримується під час підйому.
- Корос → arch сідає в середині підходу.
- Leg drive → «пасивні» ноги, Larsen ≈ звичайний жим.
- Ротаторна манжета → нестабільність плечового суглоба.

---

## Крок 3: Вибрати 1–2 вправи на 4–6 тижнів

**Локаут (трицепс, м'язовий дефіцит):**
→ Close grip bench press 3–4 × 4–8
→ JM press або board press як додаток

**Нижня точка (грудні, м'язовий дефіцит):**
→ Pause bench press 3–4 × 3–5
→ Larsen press для діагностики

**Середня фаза (передні дельти):**
→ Incline bench press 3–4 × 6–10

**Лопатки розходяться (верхня спина, позиційний):**
→ Barbell row або chest-supported row 3–4 × 6–10
→ Face pull 3 × 15–20

**Нестабільна траєкторія (стабілізація):**
→ Face pull + band pull-apart (щоденно)
→ Перевірити ретракцію лопаток у техніці

**Leg drive відсутній (стабілізація):**
→ Технічна робота: Larsen press + включення leg drive у легких підходах

**Нестабільне плечо (ротаторна манжета):**
→ External rotation 3 × 15–20
→ Face pull + band pull-apart

**Зап'ясток ламається (позиційний):**
→ Технічна корекція хвату (перший крок)
→ Wrist extension 3 × 15–20

## Швидка таблиця рішень

| Симптом | Первинна вправа | Вторинна вправа |
|---|---|---|
| Не закриваю локаут | Close grip bench | Board press |
| Нижня точка без вибуху | Pause bench | Larsen press |
| Середня фаза зупиняється | Incline bench | OHP |
| Лопатки розходяться | BB row | Face pull |
| Arch «сідає» | Farmer's carry | Planks (низький пріоритет) |
| Ноги пасивні | Технічна робота | Pause bench з акцентом |
| Гриф гуляє | Face pull | Band pull-apart |
| Плечо нестабільне | External rotation | Face pull |
| Зап'ясток ламається | Корекція хвату | Wrist extension |

## Типові помилки при виборі допоміжки

**Помилка 1: «Все після жиму»**
RDL + row + face pull + incline + pushdown — кожен день після жиму. Результат: загальна стомлюваність без специфічного ефекту.

Вибирайте 2–3 вправи максимум за тренування. Краще менше і точніше.

**Помилка 2: Занадто мала інтенсивність**
Trицепс cable pushdown на 15+ повторень не вирішить локаут-дефіцит. Специфічна сила вимагає роботи у специфічній зоні навантаження.

**Помилка 3: Ізоляція замість специфіки**
Pec deck не замінить pause bench. Triceps pushdown не замінить close grip. Ізоляція корисна, але не є основою для силового прогресу.

**Помилка 4: Зміна вправи кожні 2–3 тижні**
Адаптація до вправи — 4–8 тижнів мінімум. Якщо ви міняєте вправи щотижня — ви не даєте себе часу адаптуватись.

## Мінімальний стек «без діагнозу»

Якщо немає можливості провести відеодіагностику — мінімальний стек для більшості жимарів:

1. **Close grip bench 3 × 6** (трицепс: найчастіший обмежувач) — після жиму.
2. **Barbell row 3 × 8** (верхня спина: другий за частотою) — після жиму.
3. **Face pull 3 × 15** (плечо + ретракція: завжди корисно) — після жиму.
4. **Pause bench 1×3–5 раз на 2 тижні** (нижня точка) — замість одного звичайного сету.

Це не оптимально. Але це «почати щось» до повноцінної діагностики.

## Частота оцінки

- Через 4 тижні: чи змінились суб'єктивні відчуття у проблемній зоні?
- Через 6 тижнів: чи змінився результат у максимальних підходах?
- Через 8 тижнів: повторне відео — чи змінилась техніка в проблемній зоні?

Якщо після 8 тижнів нічого не змінилось — або діагноз неправильний, або інтенсивність/обсяг допоміжки недостатні.
