# Синтез кластеру 3: протокол підготовки верхньої частини тіла

Дев'ять постів кластеру 3 охопили всі зони та рухи. Цей пост — операційна карта для застосування.

---

## Таблиця: зона × обмеження × інструмент × час × рух

| Зона | Тест-ознака | Інструмент | Час | Пріоритет для руху |
|------|------------|-----------|-----|-------------------|
| T-spine екстензія | Wall Angel <150° | Foam Roller Extension | 3–4 хв | Bench, OHP |
| T-spine ротація | Thoracic Rotation <35° | Open Book Rotation | 2–3 хв | OHP, Row |
| Зовнішня ротація | ER <70° | Side-Lying ER + Sleeper Stretch | 3–4 хв | Bench |
| Lat / плечова флексія | Overhead Test: руки не до підлоги | Lat Stretch на стійці | 3–4 хв | OHP, Pull-up |
| RotC активація | — (превентивно) | Band Pull-Apart + Face Pull | 2–3 хв | Bench, OHP |
| Supraspinatus | Bigliani pain arc 60–120° | Full Can | 2 хв | OHP |
| Лопаткова стабілізація | Wall Slide: лікті відриваються | Scapular Pull-Up + YTW | 3–4 хв | Pull-up, Row |
| Serratus Anterior | Winging у dead hang | Serratus Push-Up | 2 хв | Pull-up |
| Зап'ясток | ER <90° у навантаженні | Wrist Extensor/Flexor stretch | 1–2 хв | Bench, OHP |

## Дерево рішень: день жиму лежачи

```
Загальна розминка (5 хв)
↓
T-spine обмежена? → ТАК → Foam Roller + Open Book (4–5 хв)
                  → НІ  → Foam Roller скорочено (2 хв)
↓
ER <80°? → ТАК → Side-Lying ER + Sleeper Stretch (4 хв)
         → НІ  → Side-Lying ER 1 × 10 (1–2 хв)
↓
Активація RotC: Band Pull-Apart + Face Pull (3–4 хв)
↓
Навантажені підходи жиму (7–10 хв)
→ ЗАГАЛОМ: 20–26 хв
```

---

## Дерево рішень: день OHP

```
Загальна розминка (5 хв)
↓
T-spine + Lat мобілізація: Foam Roller + Open Book + Lat Stretch (7–9 хв)
↓
Активація: Face Pull + Cuban Press (3–4 хв)
↓
Кор-активація: Dead Bug або Pallof Press (2 хв)
↓
Навантажені підходи OHP (8–10 хв)
→ ЗАГАЛОМ: 25–30 хв
```

---

## Дерево рішень: день підтягувань / рядів

```
Загальна розминка — гребний ергометр (5 хв)
↓
Active Lat Hang (2 хв)
↓
Лопаткова активація: Scapular Pull-Up + YTW + Band Pull-Apart (4–5 хв)
↓
Winging виявлено? → ТАК → Serratus Push-Up (2 хв)
                 → НІ  → пропустити
↓
Хват/передпліччя: Dead Hang + Wrist Rotations (2 хв)
↓
Навантажені підходи (8–12 хв)
→ ЗАГАЛОМ: 21–26 хв
```

## Мінімальний протокол (15 хв, обмежений час)

| Рух | Мінімум |
|-----|---------|
| Жим лежачи | Foam Roller 2 хв + Face Pull 2 × 12 + 3 навантажені підходи |
| OHP | Foam Roller 2 хв + Lat Stretch 2 хв + Cuban Press + 3 підходи |
| Pull-up | Active Lat Hang + Scapular Pull-Up 2 × 8 + 2 підходи |

## Що Victor відстежує у кластері 3

| Точка даних | Як використовується |
|------------|---------------------|
| Суб'єктивна оцінка плеча (1–5) до/після розминки | Виявляє ефект розминки або хронічний дефіцит |
| Час розминки | Мінімальний поріг: ≥15 хв для повного протоколу |
| Виконання чекбоксів (активація vs. пропуск) | Кореляція з показниками підходу |
| Асиметрія (Sleeper Stretch L vs. R) | Патерн: якщо права завжди гірша → сигнал sports-scientist |
| Скарги на хрускіт/клацання | Тригер для рекомендації перевірки |

## Принцип мінімальної ефективної дози (МЕД) для верхньої частини

На відміну від нижньої частини, де МЕД — 2–3 вправи per зона, верхня частина потребує двошарової підготовки:
1. **Мобілізаційний шар** (T-spine + lat + ER) — для ROM
2. **Активаційний шар** (RotC + лопатка) — для стабільності

Пропустити один шар → повний ефект не досягається навіть при правильному другому шарі. Це не «більше завжди краще» — це два обов'язкових ступені.
