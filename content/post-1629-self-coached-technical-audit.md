# Самостійний технічний аудит — 5-рухова ревізія раз на мезоцикл

Тренер на кожному тренуванні бачить те, що атлет не може побачити сам: лопаткову асиметрію ззаду, bar path збоку, зміну техніки під втомою. Self-coached атлет не має цього постійного зовнішнього ока. Але він може систематично організувати внутрішній огляд — якщо робить це структуровано і регулярно.

---

## Що таке технічний аудит і чим він відрізняється від звичайного тренування

Звичайне тренування: ціль — виконати заплановані підходи і повторення з відповідним навантаженням.

Технічний аудит: ціль — зібрати систематичну інформацію про якість рухів. Навантаження другорядне. Увага — максимальна. Обсяг — мінімальний для отримання репрезентативного зразка.

Ключова відмінність: на аудиті атлет не намагається підняти максимум. Він намагається побачити себе якомога об'єктивніше при контрольованих умовах.

---

## Покроковий протокол

### Крок 0: Підготовка (1 день до аудиту)

- Повноцінний відпочинок або легке відновлювальне тренування за день до.
- Аудит проводиться у стані повного відновлення — не після важкого тижня.
- Підготувати телефон на штативі або у надійній позиції для обох ракурсів (сагітальний і фронтальний).

### Крок 1: Виконання (1 тренування, ~60–75 хв)

Для кожного з п'яти рухів:
1. Один розминковий підхід (порожній гриф або дуже легко) — не записується.
2. Один робочий підхід (50–60% від поточного 1ПМ або типової робочої ваги), 5 повторень — **записується з двох ракурсів**.

Порядок рухів: присідання → становий підйом → жим лежачи → підтягування (або горизонтальна тяга) → жим над головою.

Між підходами — достатньо відпочинку (3–5 хвилин). Мета — кожний підхід виконується свіжим і зосередженим.

### Крок 2: Перегляд відео (того ж дня або наступного)

Переглянути кожен запис. Для кожного руху — два ракурси. Оцінювати за таксономією P0–P3:

- **P0** — критичне відхилення: потенційний ризик травматизації або суттєва структурна помилка. Усунення є пріоритетом поточного мезоциклу.
- **P1** — значуще відхилення: обмежує ефективність руху або є потенційним джерелом проблем при збільшенні навантаження. Адресується в поточному або наступному мезоциклі.
- **P2** — помітне відхилення, але не критичне. Фіксується і моніторується.
- **P3** — мінорне відхилення. Зафіксовано, але не впливає на якість чи безпеку.

### Крок 3: Запис результатів

Таблиця результатів для кожного руху:

| Рух | Тип обмежувача | P-рівень | Опис | Є в інших рухах? |
|---|---|---|---|---|
| Присідання | | | | |
| Становий підйом | | | | |
| Жим лежачи | | | | |
| Підтягування / горизонтальна тяга | | | | |
| Жим над головою | | | | |

Колонка «Є в інших рухах?» — ключова. Якщо та сама проблема з'являється в двох або більше рухах — це системний обмежувач.

### Крок 4: Ранжування пріоритетів

З усіх виявлених обмежень:
1. P0 будь-якого руху — автоматично перший пріоритет.
2. Системний обмежувач (є в ≥2 рухах) — вищий пріоритет, ніж рух-специфічний P1.
3. Серед кількох P1 — вибрати той, що найбільше впливає на основний рух (якщо є пріоритетний рух).

### Крок 5: Визначення одного системного фокусу на мезоцикл

Правило: **один фокус на мезоцикл**. Не п'ять. Не три. Один.

Паралельна робота над кількома технічними елементами одночасно розсіює увагу і ускладнює оцінку того, що працює. Послідовна корекція дає чіткий зворотний зв'язок.

Фокус може бути:
- Конкретна корекція (наприклад: «колінний трекінг у присіданні — фокус на зовнішніх ротаторах стегна»)
- Системна робота (наприклад: «торакальне розгинання — вплив на присідання, жим над головою і горизонтальну тягу»)
- Відеодіагностика конкретного параметра в кожному тренуванні (наприклад: «bar path у жимі лежачи — бічний ракурс кожні 2 тренування»)

### Крок 6: Порівняння з попереднім аудитом

Наступний аудит — через мезоцикл. Дістати попередню таблицю і порівняти:
- Чи покращився попередній P0/P1?
- Чи не з'явилися нові проблеми?
- Чи системний обмежувач зменшив своє проявлення в кількох рухах?

## Практичний висновок

Технічний аудит займає одне тренування раз на мезоцикл і 30–60 хвилин на перегляд відео. Це менше, ніж одна звичайна тренувальна сесія. Але він дає інформацію, яку жодна кількість «звичайних» тренувань без систематичного зовнішнього спостереження не дасть. Один фокус на мезоцикл, записаний і відстежений — це більше прогресу, ніж дифузна «робота над технікою» без пріоритизації.
