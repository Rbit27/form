# Sagittal Band: центратор сухожилка розгинача і механіка boxer's knuckle

Sagittal band — парна фіброзна структура, що утримує сухожилок EDC по центру над MCP суглобом. Без sagittal band сухожилок EDC зміщувався б убік при кожному скороченні — і розгинання пальця ставало б нееффективним або неможливим. Sagittal band — ця маленька структура з величезними функціональними наслідками при пошкодженні. «Boxer's knuckle» — найпоширеніша травма sagittal band у контактних спортах і важкій атлетиці. Розуміти sagittal band означає розуміти чому сухожилок EDC не «зісковзує» при флексії MCP і що відбувається, коли ця стабілізація втрачається.

---

## Анатомія: де знаходиться і з чого складається

Sagittal band розташований на рівні MCP суглобів 2–5. Це парна поперечна фіброзна структура, що обгортає капсулу MCP з дорзальної і переходить до долонного боку.

**Структурна будова:**
- Радіальна sagittal band і ульнарна sagittal band — дві смуги для кожного MCP
- Обидві прикріплюються до:
  - Сухожилка EDC (дорзально, обгортаючи його зверху і з боків)
  - Palmar plate (долонна пластинка MCP)
  - Periosteum metacarpal head

**Зв'язок із extensor expansion:**
Sagittal band є проксимальним компонентом extensor expansion (dorsal hood). Разом із lumbrical і interosseous contributions він формує повноцінний «капор» над MCP суглобом.

**Що утримує:**
Sagittal band є єдиним структурним фіксатором, що утримує EDC точно по центру (дорзально) над metacarpophalangeal суглобом. Без sagittal band сухожилок EDC є нестабільним і при активному навантаженні зміщується у міжпальцеву щілину.

---

## Функція: центрація і передача зусилля

**Первинна функція sagittal band:**
Утримання сухожилка EDC по центральній осі metacarpal head при всіх положеннях MCP від максимального розгинання до максимального згинання.

**Механіка центрації:**
- При **розгинанні MCP**: sagittal band знаходиться у нейтральному стані; EDC тягне проксимально → MCP розгинається
- При **згинанні MCP**: палець складається у кулак → metacarpal head «виходить вперед» дорзально → sagittal band натягується, утримуючи EDC від зісковзування до бічної щілини

**Передача зусилля lumbricales:**
Sagittal band також є місцем долучення lumbricales та interossei до extensor expansion. Lumbricales, скорочуючись, тягнуть lateral bands дорзально → через sagittal band і extensor expansion → розгинання IP суглобів. Ця передача зусилля від долонного м'яза до дорзальної поверхні пальця є архітектурним дивом кисті.

## Пошкодження sagittal band: класифікація і механізм

**Механізм травми:**
1. **Пряма травма:** удар кулаком (бокс, бойові мистецтва) → sagittal band ушкоджується безпосередньо при контакті MCP з твердою поверхнею
2. **Гіперфлексія MCP:** екстремальне згинання з навантаженням → sagittal band перерозтягується
3. **Дегенеративний розрив:** при ревматоїдному артриті (синовіт ерозує sagittal band → прогресуюча ульнарна девіація пальців)

**Класифікація Рунянського (Rayan-Murray):**
| Ступінь | Опис |
|---------|------|
| I | Набряк і больова реакція, sagittal band цілий |
| II | Часткова сльоза — сухожилок EDC зміщується при активному русі, але повертається у центральну позицію |
| III | Повна сльоза — сухожилок EDC залишається зміщеним, не повертається спонтанно |

**Найчастіше залучений палець:** 3-й (середній) — через його центральне положення при ударі кулаком і найменший захист juncturae tendinum.

## Boxer's knuckle: клінічна картина

«Boxer's knuckle» — термін, що охоплює спектр травм MCP від контузії м'яких тканин до повного розриву sagittal band і капсули.

**Що болить і чому:**
При boxer's knuckle больова точка — дорзальна поверхня MCP sugloba («кісточка»). При пальпації — різкий біль саме над metacarpal head. При активному згинанні кулака спостерігається:
- Клацання або відчуття ковзання (step-off sign) при зміщенні EDC
- Послаблення сили розгинання при спробі «вдарити» — EDC зміщений і не передає сили ефективно

**Тест нестабільності EDC:**
1. Рука у нейтральній позиції
2. Спостерігайте або пальпуйте сухожилок EDC над MCP
3. Повільно стискайте кулак (флексія MCP)
4. При розриві sagittal band: EDC «провалюється» до ульнарної або радіальної щілини між metacarpals → пальпується зісковзування

**Ульнарна vs радіальна сльоза:**
- Ульнарна сльоза sagittal band → EDC зміщується радіально (до 2-го пальця)
- Радіальна сльоза → EDC зміщується ульнарно
- Ульнарна сльоза 3-го пальця є найпоширенішою для боксерів — EDC «йде» між 3-м і 4-м пальцями

---

## Sagittal band при ревматоїдному артриті: окрема патологія

При РА синовіт MCP суглоба ерозує sagittal band зсередини. Прогресування:
1. Ранній РА: синовіт → набряк → сагітальні смуги розтягнуті
2. Хронічний РА: прогресуюча ерозія → всі EDC зміщуються ульнарно (ульнарна девіація пальців)
3. Пізній РА: класична «ульнарна drift» — пальці 2–5 відхилені ульнарно, EDC підвивнуті між metacarpals

Ця деформація є патогномонічним знаком РА кисті. Хірургічна корекція включає централізацію EDC і реконструкцію sagittal band.

---

## Лікування розриву sagittal band

**Консервативне лікування (ступінь I–II):**
- Іммобілізація: palmar splint або «buddy taping» суміжного пальця
- MCP у позиції ~15–20° флексії (зменшує натяг sagittal band)
- Тривалість: 4–6 тижнів
- Відновлення до повного навантаження: 8–12 тижнів
- Спортивний тейп дозволяє більш раннє повернення до тренувань при частковому пошкодженні

**Хірургічне лікування (ступінь III або неефективне консервативне):**
- Пряма реконструкція sagittal band (зшивання)
- Tenodesis (використання частини juncturae tendinum як аугментаційного матеріалу)
- Термін після операції до спорту: 3–4 місяці

## Sagittal band у силових видах спорту

**Ризик-профіль:**
Sagittal band травми у силовому тренінгу — рідкість при «класичному» підйомі залізом. Ризик вищий у:
- **Важкій атлетиці (Olympic lifting):** при «підбитті» кисті у rack position (clean) — MCP раптово згинається у повне згинання з навантаженням; radial sagittal band 3-го пальця під ризиком
- **Powerlifting (sumo deadlift):** при «врізанні» суглобів у конкурентній позиції — менш виражений ризик, але існує
- **Gymnastic grip на crossfit:** grip failure + raптова флексія MCP під навантаженням

**Тейпінг профілактика:**
При поверненні до тренувань після boxer's knuckle — тейпінг MCP суглоба у позиції незначної флексії захищає sagittal band від повторної травми при формуванні фіброзного рубця (мінімум 6–8 тижнів після травми).

**Знак повернення до тренувань:**
Безболісне стискання кулака + тест нестабільності EDC (відсутнє зісковзування при флексії) + відсутність болю при максимальному ізометричному хваті → можна повертатись до навантажень з тейпом.

## Sagittal band і extensor expansion: ієрархія структур

Розуміння sagittal band у контексті extensor expansion важливе для загальної картини:

```
Extensor expansion = {
  Sagittal bands (проксимально)   — центрація EDC над MCP
  Dorsal hood (над MCP)           — місце долучення lumbricales, interossei
  Central slip (дистально)        — розгинання PIP
  Lateral bands (дистально)       — розгинання DIP
  Terminal tendon (кінець)        — прикріплення до distal phalanx
}
```

Sagittal band є «воротами» extensor expansion — він контролює, чи правильно EDC позиційований для передачі зусилля в усі дистальні компоненти. Без нього все, що нижче, функціонує неефективно.

---

## Практичний алгоритм самооцінки sagittal band

**При болю в MCP після удару або навантаження:**
1. Пальпуйте дорзальну поверхню MCP — чи є локальна точка болю над «кісточкою»?
2. Повільно стискайте кулак — чи відчуваєте клацання або відчуття ковзання?
3. Спробуйте силовий хват — чи відчуваєте слабкість або pain при повному стисканні?
4. Спостерігайте положення EDC при повному кулаці — чи зміщений він убік від центру MCP?

Будь-який з пунктів 2–4 → можлива травма sagittal band → консультація хірурга кисті перед поверненням до навантажень.

## Підсумок

Sagittal band — парна поперечна фіброзна структура над MCP суглобами, що утримує EDC по центральній осі metacarpal head при всіх положеннях пальця. Без sagittal band EDC зміщується у міжпальцеву щілину при флексії MCP, втрачаючи ефективність розгинання. Пошкодження sagittal band (boxer's knuckle) класифікується за Rayan-Murray (I–III): часткова сльоза дає динамічну нестабільність EDC при стисканні, повна — постійне зміщення. Ульнарна сльоза 3-го пальця є найпоширенішою у боксерів. Консервативне лікування (4–6 тижнів іммобілізації) достатнє для ступенів I–II; III ступінь потребує хірургічної реконструкції. При РА sagittal band ерозується і спричиняє прогресуючу ульнарну девіацію — патогномонічну ознаку РА кисті. Для атлета: будь-яке клацання або відчуття ковзання в MCP після травми — показання до обстеження до повернення до повного навантаження.
