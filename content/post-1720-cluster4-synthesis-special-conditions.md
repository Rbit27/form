# Синтез кластеру 4: матриця особливих умов

Дев'ять постів кластеру 4 охопили чотири типи особливих умов. Цей пост — операційна карта.

---

## Матриця умов × протокол × пріоритет

| Умова | Тривалість розминки | Ключовий пріоритет | Головна помилка |
|-------|--------------------|--------------------|-----------------|
| Стандарт (залізо, час є) | 20–25 хв | Повна мобілізація + активація | Поспіх |
| Обмежений час (10–15 хв) | 10–15 хв | Ієрархія: загальна → пріоритетна зона → навантажені підходи | Все потроху — нічого повністю |
| 5 хвилин | 5 хв | Загальна (2 хв) + одна вправа + 3 підходи | Пропустити загальну розминку |
| Змагальний день | 25–35 хв (від першого підходу) | PAP-підхід за 8–12 хв до виходу | Продовжувати після PAP |
| Подорож | 10–15 хв | Тіловаговий потік: загальна + мобілізація + активація | Пропустити повністю («все одно тіловагове») |
| Вдома | 12 хв | Ритуал запуску + функціональні вправи під тип тренування | Старт без переходу |
| Поганий стан / недосип | 15–20 хв | Діагностичний RPE у навантажених підходах | Ігнорувати RPE і форсувати план |

## Дерево рішень: вибір протоколу

```
Яка умова сьогодні?
│
├─ Стандарт + час → Повний протокол (кластер 1–3)
│
├─ Обмежений час
│   ├─ 10+ хв → 10-хвилинний протокол (пост 1713)
│   └─ <10 хв → 5-хвилинний протокол (пост 1712)
│
├─ Змагальний день
│   ├─ Жим → пост 1715
│   └─ Присідання / Становий → пост 1716
│
├─ Без залу
│   ├─ Подорож / готель → пост 1717
│   └─ Вдома → пост 1718
│
└─ Поганий стан / недосип → пост 1719 (діагностичний протокол)
```

## Принципи, спільні для всіх умов

### 1. Загальна розминка не вирізається першою

У будь-якій умові — якщо є 3 хвилини на загальну розминку, вони витрачаються на неї. Підняти температуру тіла є фундаментом. Без нього ефективність решти розминки падає непропорційно.

### 2. Навантажені підходи (або їх тіловаговий еквівалент) — незмінна константа

Нервово-м'язова активація для конкретного руху не замінюється мобілізацією. Навіть у 5-хвилинному протоколі — три підходи штанги (або тіловагових вправ).

### 3. Скорочення — це ієрархія, не рівномірне урізання

Першими вирізаються профілактичні елементи для зон без поточних обмежень. Останніми — загальна розминка і навантажені підходи.

## Що Victor відстежує в особливих умовах

| Ситуація | Дані, які фіксуються |
|----------|---------------------|
| Обмежений час | Фактична тривалість розминки vs. RPE першого підходу |
| Змагальний день | Час PAP-підходу відносно виходу; RPE опенера |
| Подорож / вдома | Чи виконана розминка? Тип (повна / скорочена) |
| Поганий стан | Суб'єктивний стан + RPE 65% → рішення по плану |

Паттерн з часом: Victor виявляє кореляцію між пропущеними/скороченими розминками і RPE першого підходу. Не каже що робити — показує що є.

## Кластер 4: COMPLETE

Пости 1711–1720 охоплюють чотири типи особливих умов:
- **Обмежений час** (1711–1713): ієрархія скорочення, 5 і 10-хвилинні протоколи
- **Змагальний день** (1714–1716): стратегія + конкретні схеми для жиму, присідання, станового
- **Без залу** (1717–1718): протоколи для подорожі і дому
- **Поганий стан** (1719): діагностичний підхід до авторегуляції
