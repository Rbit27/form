# Тижневий аудит: як читати чотири потоки даних за тиждень

Щоденне рішення — тактика. Тижневий огляд — стратегія.

Щоденна зона говорить «що робити сьогодні». Тижневий аудит говорить «чи ми рухаємося у правильному напрямку» і «чи не накопичуємо проблему».

---

## Коли робити тижневий аудит

Раз на тиждень — в одну і ту саму добу (зазвичай неділя або понеділок). 10–15 хвилин.

Не під час або одразу після важкого тренування — стан впливає на об'єктивність.

---

## П'ять запитань тижневого аудиту

### 1. Яким був тижневий тренд HRV?

Дивись не на вчорашній показник, а на 7-денну криву.

- **Стабільна або зростаюча крива** → організм адаптується, навантаження посильне
- **Постійно знижена крива протягом тижня** → сигнал перевантаження або нетренувального стресу
- **Різкий одиночний провал** + повернення → ізольована подія (пізній вечір, стрес, алкоголь), не системна

**Питання для запису:** Скільки зелених / жовтих / червоних днів по HRV цього тижня?

---

### 2. Яким був тижневий тренд сну?

Порівняй суму годин сну цього тижня з попереднім.

- Тижнева тривалість нижча на >5 год порівняно з попереднім тижнем → борг, що кумулюється
- Стабільна тривалість + стабільна суб'єктивна якість → базова відновлювальна функція в нормі
- Хороша тривалість + низька суб'єктивна якість 2+ рази → зверни увагу на якість (стрес, переривчастість)

**Питання для запису:** Середня тривалість за тиждень? Кількість ночей з суб'єктивною оцінкою ≤5?

---

### 3. Яка середня суб'єктивна оцінка за тиждень?

Підсумуй всі щоденні оцінки і порівняй із попереднім тижнем.

- Середнє ≥7 → хорошій рівень суб'єктивного стану
- Середнє 5–6 → прийнятно, але слідкуй за трендом: якщо два тижні поспіль — розбирайся з причиною
- Середнє ≤4 → сигнал. Чи не накопичується перенавантаження?

**Особлива ознака:** якщо суб'єктивне систематично нижче, ніж «мало б бути» при нормальних HRV і сні — перевір нетренувальні стресори (робота, сон, харчування, вірус).

**Питання для запису:** Середнє суб'єктивне за тиждень? Тренд вгору, вниз, стабільно?

---

### 4. Яким був тижневий об'єм і ACWR?

- Тижневий тоннаж відносно попереднього тижня: ріст, стабільно, зниження
- ACWR: в нормальному коридорі 0.8–1.3 чи вийшов за межі?

**Типові помилки тижня по об'єму:**
- Пропуск 2+ тренувань → ACWR < 0.8 → детренінг
- Спроба «надолужити» пропущені тренування за 1–2 дні → spike → ACWR > 1.5
- Поступове щотижневе зростання без деload → кумулятивне перевантаження через 4–6 тижнів

**Питання для запису:** Тижневий тоннаж? ACWR? Чи відповідає це плановому прогресу?

---

### 5. Чи є патерн між зонами і якістю тренування?

Це аналіз за 4+ тижні — не за один тиждень. Але записуй зараз, щоб потім порівняти.

- У жовті дні тренінг йде гірше чи я перебільшую вплив зони?
- У зелені дні я дійсно тренуюсь краще за планом?
- Скільки червоних днів на місяць і чи є в них спільний контекст?

---

## Формат тижневого запису (5 хвилин)

```
Тиждень: [дати]

HRV trend: ___ зел / ___ жов / ___ черв
Середня тривалість сну: ___ год
Суб'єктивне середнє: ___ /10
Тижневий тоннаж: ___ кг (±___% від попереднього)
ACWR: ___

Загальна оцінка тижня: зелено / жовто / червоно
Головний висновок: ________________________
Зміна на наступний тиждень (якщо є): ________________________
```

---

## Що робити з висновком

Тижневий аудит — не для самокатування і не для підрахунку балів. Він для одного: побачити ранні ознаки проблеми і відреагувати на 1–2 тижні раніше, ніж вона стане серйозною.

Один жовтий тиждень — норма. Два жовтих тижні поспіль — переглянь план. Три — зроби deload незалежно від того, хочеться чи ні.
