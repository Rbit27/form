# Типи м'язових волокон і тренінг: що ти можеш і чого не можеш змінити

У більшості розмов про програмування типи волокон з'являються у двох форматах: або «ти повинен знати свій тип волокон» — зазвичай передує продажу якогось тесту або добавки — або «це не має значення, просто піднімай важче». Обидва спрощення.

Тип волокна має значення. Але не так, як зазвичай описується.

---

## Базова класифікація: що таке Type I і Type II

М'язові волокна класифікуються за ізоформою міозину (myosin heavy chain, MHC) — це білок, що утворює «голівки» миозинових ниток і безпосередньо визначає швидкість скорочення і метаболізм волокна.

**Type I (повільні, «slow-twitch»)**
- Ізоформа: MHC-I (β-MHC)
- Швидкість скорочення: низька
- Пікова сила: нижча ніж у Type II
- Відносини до втоми: висока стійкість (oxidative, аеробний метаболізм домінує)
- Щільність мітохондрій: висока
- Капілярна мережа: густа
- Колір: червоний (міоглобін)
- Роль: витривалість, підтримка пози, тривалі субмаксимальні зусилля

**Type IIa (швидкі, «fast-twitch oxidative-glycolytic»)**
- Ізоформа: MHC-IIa
- Швидкість скорочення: висока
- Пікова сила: вища ніж Type I
- Стійкість до втоми: помірна (hybrid metabolism)
- Адаптивність: найбільш пластичні — зміщуються до Type I або IIx залежно від тренінгу
- Роль: більшість силових і гіпертрофійних рухів у середньому діапазоні інтенсивності

**Type IIx (швидкі, «fast-twitch glycolytic»)**
- Ізоформа: MHC-IIx (раніше називались IIb у людей, хоча IIb-ізоформа у людей практично відсутня)
- Швидкість скорочення: максимальна
- Пікова сила: максимальна
- Стійкість до втоми: дуже низька
- Роль: спринт, вибухові зусилля, максимальні підйоми

Перехідна зона між типами: багато волокон є гібридними — містять суміш MHC-ізоформ. Чиста класифікація — аналітична зручність, а не ідеальне відображення реальності м'язових пучків.

## Генетичний компонент: що визначено, а що — ні

Пропорція Type I до Type II у кожному м'язі значною мірою визначена генетично і відносно стабільна у дорослої людини. Дослідження на однояйцевих близнюках (Komi та Karlsson, 1979; Simoneau і Bouchard, 1995) показують: ~45% варіативності складу волокон визначається генетикою, ~45% — тренінгом і середовищем.

**Що генетика реально визначає:**
- Базову пропорцію Type I / Type II у кожному м'язі (різна для різних м'язів одного тіла)
- «Потенційну стелю» нейром'язової потужності та витривалості
- Швидкість адаптації до різних типів тренінгу

**Що генетика не визначає:**
- Твій результат. Elite powerlifters мають не виключно Type II-домінантний склад — є чемпіони з помірним Type I-відсотком, які компенсують це технікою, об'ємом і нейром'язовою ефективністю
- Гіпертрофійний потенціал Type I волокон (вони гіпертрофують — просто через інші сигнальні шляхи і повільніше при низькому об'ємі)
- Здатність до адаптації Type IIa (найбільш пластичний тип — зміщується суттєво при правильному тренінгу)

Детерміністична рамка — «у тебе більше Type I, тому ти ніколи не будеш потужним» — ні наукою не підкріплена, ні практично марна. Більш продуктивний підхід: думати про тип волокна як про характеристику робочого матеріалу, а не як про вирок.

## Принцип розміру і рекрутування: хто активується коли

Henneman's Size Principle (1957) — фундамент розуміння рекрутування:

Мотонейрони рекрутуються у порядку збільшення розміру: спочатку менші (іннервують Type I), потім більші (іннервують Type II). Таким чином:

- **Легкі ваги:** переважно Type I активовані
- **Важчі ваги:** Type I + поступово Type IIa
- **Максимальні зусилля (>85-90% 1RM):** усі типи волокон залучені — і Type I, і IIa, і IIx

Наслідок для програмування: тренування з важкими вагами (>80% 1RM) рекрутує і тренує весь спектр волокон. Легкі ваги до відмови також можуть рекрутувати Type II — але тільки ближче до відмови, коли Type I виснажені і нервова система змушена «підключати» більші мотонейрони.

Schoenfeld et al. (2017) — метааналіз: при порівнянному обсязі (підходи × повторення × вага) важкі і легкі протоколи (6 vs 25 повторень) давали подібну гіпертрофію у загальному об'ємі м'яза. Але є нюанс — до якого повернемося.

## Тренінг-специфічна адаптація волокон

### Чи може Type II перейти у Type I і навпаки?

Повний перехід Type I ↔ Type II (зміна нейрональної іннервації) — у дорослих людей не підтверджений. Але є інший, підтверджений феномен: **зміщення в межах Type II** — між IIx і IIa.

**Ефект тренінгу на Type IIa/IIx:**

При силовому і особливо вибуховому тренінгу відбувається зміщення IIx → IIa. Це виглядає як парадокс («силовий тренінг робить волокна менш «швидкими»?»), але логіка така: IIa — більш пластичний hybrid-тип, краще підходить для повторних силових зусиль з відновленням між підходами.

Цікавий феномен: detrained individuals (або ті, хто починає тренінг після паузи) мають більший відсоток IIx, ніж тренований атлет. Після початку тренінгу IIx зменшується, IIa зростає.

### Hypertrophy: Type I vs Type II

Type II волокна мають вищий гіпертрофійний потенціал при стандартних протоколах. Але:

- Type I також гіпертрофують — особливо при специфічних умовах (тривалі сети, підвищений TUT, метаболічний стрес)
- Mitchell et al. (2012): 30% 1RM до відмови vs 80% 1RM — подібна загальна гіпертрофія, але розподіл між типами відрізнявся: нижчі ваги до відмови краще гіпертрофували Type I

Для практичного атлета: якщо твої м'язи Type I-домінантні (наприклад, vastas lateralis у ендurance-спортсменів), то виключно важкий powerlifting-протокол може недостатньо стимулювати їх гіпертрофію. Включення вищого rep-range (15–20 повторень) у программу може додати стимул для Type I.

## Практичне програмування: що реально змінює тип

Те, що реально впливає на «функціональний тип» волокон у тренованого атлета:

**1. Інтенсивність (% 1RM):**
Важкі ваги (>80%) максимально рекрутують Type II. Обов'язково для розвитку їх сили і потужності. Без регулярного тренінгу у цьому діапазоні Type IIx/IIa не отримують специфічного адаптаційного стимулу.

**2. Швидкість виконання (velocity):**
Зусилля з максимальною навмисною швидкістю (навіть якщо реальна швидкість штанги низька через важку вагу) — один з найефективніших сигналів для Type II. Weakened MHC-IIa motor units рекрутуються при максимальній командній частоті. Це підтверджує Velocity-Based Training (VBT) — попередній пост.

**3. Rep range і втома:**
- 1–5 повторень: переважно нейром'язова адаптація, Type IIx рекрутування
- 6–12 повторень: класичний гіпертрофійний діапазон, Type IIa-dominant
- 15–30 до відмови: включає Type I гіпертрофію, метаболічний стрес

**4. Час під напругою (TUT):**
Довший TUT при помірній вазі більше навантажує Type I. Але якщо загальний об'єм низький — гіпертрофійний стимул може бути недостатнім навіть для Type I.

**5. Частота:**
Type II волокна відновлюються повільніше від нейром'язового стресу. Частота для чисто силових/потужністних стимулів — 2–3 рази на тиждень на м'яз дозволяє адекватне відновлення IIx.

## Тестування типу волокон: чи потрібно і чи реалістично

Єдиний точний метод — м'язова біопсія (голчаста або відкрита). Ні не варто, ні не реалістично для більшості атлетів.

**Польові тести (наближені):**

«Rep max test»: виконай вправу на 80% від 1RM до відмови. Умовна інтерпретація:
- <7 повторень — ймовірна Type II домінанта у цьому м'язі
- 7–12 — змішаний склад
- >12 — ймовірна Type I домінанта

Але цей тест перемішує тип волокна з нейром'язовою ефективністю, тренованістю і витривалістю нервової системи. Він вказує на «тренінгову характеристику» м'яза, а не на чистий тип волокна. Для практичного програмування цього достатньо.

**Що варто взяти з тесту:** якщо при 80% 1RM ти без проблем виконуєш 15+ повторень у присіданні — є сенс включити вищий rep-range для квадрицепсу. Якщо «здуваєшся» на 5 повтореннях — Type II-орієнтований стимул вже є, і додаткові 20+ rep-сети не є пріоритетом.

## Як FORM враховує волоконну специфіку

Тип волокна не вимірюється напряму — ні FORM, ні більшість систем без біопсії. Але прогресія і варіантність навантаження у FORM-алгоритмах враховує «функціональну поведінку» м'язів атлета:

**Velocity tracking:** якщо bar speed при субмаксимальній вазі падає раніше ніж очікується за моделлю — це може вказувати на відносну Type II-слабкість. Протокол коригується убік нижчого rep-range і вищої інтенсивності.

**Rep-at-RPE калібрація:** якщо атлет систематично виконує більше повторень на заданому RPE ніж прогнозує модель — це узгоджується з Type I-домінантним профілем. Програма може включати більший об'єм у вищому rep-range.

Це не «тест типу волокон» — це адаптація програмування під поведінку конкретного атлета. Результат схожий.

## Що робити

**Для intermediate-атлета з гіпертрофійною ціллю:**

- Тримай базові компаунди у діапазоні 4–8 повторень (стимул Type IIa)
- Аксесуарні — 10–15 повторень (hybrid стимул)
- Раз на місяць — 20+ rep фіналізуючі сети на великі м'язи (Type I стимул)
- Не нехтуй швидкістю виконання — «навмисно швидко» навіть при помірних вагах рекрутує Type II

**Якщо помічаєш, що конкретний м'яз «не відповідає» стандартному протоколу:**

1. Виконай rep max test при 80% 1RM для цієї вправи
2. Якщо >12 повторень — експериментуй з вищим rep-range і більшим TUT
3. Якщо <5 повторень — можливо, нейром'язова ефективність і не тип волокна є обмеженням

**Не роби:**
- Не оптимізуй програму виключно «під свій тип волокон» без реального тестування
- Не фреймуй свій склад волокон як ceiling — адаптивний потенціал атлета ширший, ніж вихідна пропорція типів

## Що варто забрати

Тип волокон — один із чинників серед багатьох. Він впливає на оптимальний rep-range, на швидкість адаптації і на те, який тренінг дасть найкращий відгук. Але не визначає стелю твого результату.

Найбільш пластичний тип (IIa) відповідає на широкий спектр стимулів. Type I гіпертрофує при правильних умовах. Type IIx рекрутується при максимальних зусиллях і адаптується до tренінгу.

Програма, що включає весь rep-range (тяжкий базовий тренінг + гіпертрофійний об'єм + рідкі висококонтактні «burnout» сети) стимулює весь спектр волокон. Це краща стратегія, ніж оптимізація під уявний «домінантний тип».
