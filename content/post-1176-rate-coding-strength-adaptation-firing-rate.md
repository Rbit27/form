# Rate coding: як ЦНС підвищує силу без додаткових волокон

## Після рекрутування — частота

Принцип Хеннемана (пост 1152) описує першу фазу нейронної регуляції сили: рекрутування рухових одиниць від низькопорогових до високопорогових у порядку збудливості. При 85–90% MVC практично всі рухові одиниці вже рекрутовані. Що відбувається далі, коли потрібно ще більше сили?

Відповідь — **rate coding**: збільшення частоти електричних розрядів уже активованих рухових одиниць. Вища частота → більш злите скорочення (tetanus) → вищий силовий внесок кожного волокна.

Це не теоретична конструкція. Це нейрофізіологічний механізм із прямими практичними наслідками для того, як потрібно тренуватись, щоб виробити максимальну силу.

---

## Як виглядає rate coding у числах

Одиночне скорочення (twitch) рухової одиниці дає пікову силу за ~50–100 мс і потім розслаблюється. При послідовних розрядах із частотою вище часу розслаблення волокна ефекти підсумовуються: часткова summation при 15–30 Гц, повний тетанус при 50–80+ Гц залежно від типу волокна.

IIx-волокна (швидкі) мають коротший час скорочення і тому потребують вищих частот для tetanus. IIa-волокна — проміжні. I-тип — найповільніші, тетанус при нижчих частотах.

Практично: для максимального силового виходу у швидких волокнах ЦНС повинна генерувати дуже високочастотні розряди — і ця здатність є **тренованою**.

---

## Нейронна адаптація rate coding з тренінгом

Carroll et al. (2002) порівнювали пікові частоти розрядів рухових одиниць у тренованих і нетренованих осіб при максимальних ізометричних скороченнях. Тренована група демонструвала вищі пікові частоти при рівному відсотку MVC — нейронна адаптація, що не залежить від розміру м'яза.

10 тижнів силового тренінгу дали значиме підвищення rate coding при субмаксимальних скороченнях (Carroll et al., 2011). Механізм: зміни в іонних каналах мотонейронів, підвищена persistent inward current (PIC), знижена після-потенційна гіперполяризація — нейрон може «стріляти» частіше без підвищення порогу збудження.

## Які тренувальні стимули розвивають rate coding

### Важка робота при 85–97% 1RM

Для того щоб ЦНС «тренувала» здатність генерувати пікові частоти розряду, вона повинна виконувати максимальні або близькі до максимальних вольові зусилля. Робота при 65–75% 1RM — недостатній стимул для розвитку пікового rate coding, навіть якщо виконується до відмови. Відмова при легшій вазі через накопичення метаболітів у волокнах типу I і IIa, але пік частоти у вже рекрутованих IIx-одиницях при цьому менший.

Важкі одиночні (singles) і двійки (doubles) при 90–97% 1RM — специфічний стимул для пікового rate coding. Субмаксимальна важка робота (5–6 повторів при ~85%) теж розвиває rate coding, але не так специфічно, як ближчі до максимуму зусилля.

### Cluster sets як метод зберігання якості при важкій роботі

Cluster sets (відпочинок 15–30 сек між повтореннями всередині підходу) дозволяють виконати більше технічно якісних повторень при 90%+ без деградації паттерну через накопичення периферичної втоми. Нейронна якість кожного повторення (пік rate coding на вольовий поштовх) залишається вищою.

### Балістична робота і розвиток початкового rate coding

Швидкі балістичні рухи (jump squat, medicine ball throw) розвивають здатність до швидкого нарощування частоти розряду в перші 50–100 мс руху — initial rate coding. Sale (1988) показав, що тренування з акцентом на швидкість руху збільшує slope частоти розряду, а не тільки пікове значення. Це важливо для спортивних рухів, де сила розвивається у коротких часових вікнах.

## «Grease the groove» і його нейронні межі

Pavel Tsatsouline популяризував підхід «grease the groove»: часті субмаксимальні підходи (50–60% RM) без втоми для тренування нейронного паттерну. Це має нейронне обґрунтування для початкового рекрутування і координаційних адаптацій.

Але для **rate coding** субмаксимальна субмаксимальна робота є недостатнім стимулом. Нейрон тренує здатність до пікової частоти розряду лише тоді, коли від нього вимагається піковий розряд. Метод GtG є доповненням до важкої роботи, а не її заміною, якщо мета — максимальна сила.

---

## Чому rate coding є «невидимою» адаптацією

Гіпертрофія видима: м'яз збільшується в розмірі, зміни відслідковуються через обводи або DEXA. Rate coding не видний без електроміографії і не відчувається як специфічна адаптація.

Але він пояснює феномен, знайомий тренованим атлетам: після правильно виконаного тестового тижня або після деловантаження вага, яка здавалась важкою, стає «легшою» — не тому що м'яз виріс (для гіпертрофії часу не вистачило), а тому що ЦНС ефективніше використовує наявний м'язовий потенціал.

Rate coding разом із VA (пост 1177) є механізмом «нейронного піку» — короткострокового підвищення силового виходу без структурних змін.
