# Відеодіагностика станового підйому: два ракурси і протоколи читання

Відеозйомка — єдиний реалістичний інструмент самодіагностики техніки для атлета, який тренується без тренера. Без відео ви оцінюєте рух через пропріоцептивне відчуття, яке систематично розходиться з реальністю під важкими вагами.

---

## Два ракурси, два протоколи

### Ракурс 1: Збоку (сагітальна площина)

**Де ставити камеру:** На рівні колін або стегон, збоку від атлета — 90° до напрямку погляду. Відстань 2–3 метри.

**Що читати в цьому ракурсі:**

| Момент підйому | Що перевіряємо |
|---|---|
| Стартова позиція | Штанга над серединою стопи; лопатки над штангою |
| До відриву | Нейтральний поперек; кут тулуба |
| Відрив | Таз і плечі рухаються вгору разом або таз вирвався вперед |
| Перший пул (до колін) | Штанга ковзає по гомілці або відходить від ноги |
| Другий пул (після колін) | Таз рухається вперед, плечі заходять за вертикаль штанги |
| Lockout | Тіло вертикальне; немає гіперекстензії назад |

### Ракурс 2: Спереду або ззаду (фронтальна площина)

**Де ставити камеру:** Прямо спереду або ззаду, на рівні колін. Відстань 2–3 метри. У сумо — ззаду інформативніше.

**Що читати в цьому ракурсі:**

| Момент підйому | Що перевіряємо |
|---|---|
| Стартова позиція | Симетрія (одне плече нижче іншого, одна нога висунута вперед) |
| Відрив | Чи штанга не нахиляється одним кінцем вниз (асиметричний хват або слабкість) |
| Перший пул | Колінний трекінг: коліна рухаються у напрямку носків або колапсують всередину |
| Lockout | Симетрія фінальної позиції |

## Пріоритети при читанні відео

Типова помилка при аналізі свого відео — намагатися виправити все одразу. Ефективніший підхід: ієрархія пріоритетів.

**P0 (виправляти зараз, знижувати вагу якщо потрібно):**
- Повна поперекова флексія на старті або під час відриву
- Штанга різко відходить від ноги при відриві

**P1 (виправляти протягом 4–6 тижнів):**
- Ранній підйом таза (hips rise too fast)
- Значна асиметрія у відриві або першому пулі

**P2 (виправляти в наступному мезоциклі):**
- Незначний надлишок нахилу у lockout
- Невеликий вальгус колін у сумо (залежно від амплітуди)

**P3 (моніторити, але не коригувати активно):**
- Природний нахил тулуба через антропометрію
- Округлення грудного відділу при важких підходах

## Практичний протокол зйомки

1. Знімайте робочий сет або підхід при 80–85% від максимуму — при мінімальній вазі техніка не відображає реальних важких умов.
2. Один телефон збоку — основний ракурс.
3. Якщо є помічник — другий ракурс спереду одночасно.
4. Перегляньте на швидкості 0.25x або 0.5x. При нормальній швидкості деталі не читаються.
5. Зафіксуйте один-два пріоритетних пункти для корекції. Не п'ять.

---

## Ключове обмеження самодіагностики

Відео дає картинку, але не завжди дає причину. Якщо ви бачите, що таз піднімається раніше плечей — це можна пояснити кількома причинами (слабкість квадрицепсів, неправильне положення старту, неоптимальний кут таза). Обрати правильну корекцію без розуміння причини — означає гадати. Post-1589 дає матрицю для переходу від симптому до причини.
