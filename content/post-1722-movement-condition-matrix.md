# Матриця рух × умова: операційна таблиця вибору протоколу

Блок 1681–1730 описав кожен рух і кожну умову окремо. Цей пост об'єднує їх в одну операційну таблицю.

Логіка проста: перед тренуванням у тебе є рух і умова. Перетин у таблиці дає протокол. Не «загальні рекомендації» — конкретний вибір.

---

## Як читати таблицю

**Рухи** (рядки): присідання, становий підйом, жим лежачи, підтягування/горизонтальна тяга, жим стоячи.

**Умови** (стовпці):
- **Стандарт** — залізний зал, 20+ хвилин на розминку, нормальний стан
- **10–15 хв** — обмежений час, але є штанга
- **5 хв** — критично обмежений час
- **Змагальний день** — перший підхід важливіший за все
- **Без залу** — подорож, готель, вдома
- **Поганий стан** — недосип, хвороба, висока втома

Протокол у кожній клітині — мінімально необхідний. Якщо є час зробити більше — роби. Якщо ні — цього достатньо.

## Матриця: присідання

| Умова | Протокол |
|-------|----------|
| Стандарт (20+ хв) | Загальна 3 хв → гомілковостопна дорсифлексія 2 хв → кульшові CARs 2 хв → goblet squat з паузою 2×5 → навантажені підходи 30–85% |
| 10–15 хв | Загальна 2 хв → goblet squat з паузою 1×8 → навантажені підходи від 40% |
| 5 хв | Загальна 90 с → 3 навантажені підходи (40%, 55%, 70%) |
| Змагальний день | Повний протокол + PAP-підхід 90–95% за 8–12 хв до виходу |
| Без залу | Загальна 2 хв → bodyweight squat 2×10 → сплітт-присідання 1×8 → стрибки або пульсації |
| Поганий стан | Стандартний протокол → діагностика RPE на 65%: якщо ≥ 8 → знижуй робочу вагу на 5–7% |

**Чому так:**
- Присідання вимагає трьох зон одночасно — кульшовий, гомілковостопний, люмбопельвічний (пост 1691–1700). При обмеженому часі вирізається мобілізація гомілковостопного (якщо немає відомих обмежень), зберігається кульшова підготовка у формі goblet.
- 5-хвилинний протокол без мобілізації — прийнятно, якщо навантажені підходи починаються достатньо легко (40%, не 60%).
- Поганий стан: діагностичний RPE саме на 65% тому, що це вага, яка не потребує зусиль при нормальному стані. Якщо вона «важка» — сигнал реальний.

## Матриця: становий підйом

| Умова | Протокол |
|-------|----------|
| Стандарт (20+ хв) | Загальна 3 хв → кульшові CARs 2 хв → розгиначі спини без стомлення (bird-dog 2×5) → RDL з мінімальною вагою 2×5 → навантажені підходи 30–85% |
| 10–15 хв | Загальна 2 хв → кульшові CARs 1×5 → навантажені підходи від 40% |
| 5 хв | Загальна 90 с → 3 навантажені підходи (40%, 55%, 70%) |
| Змагальний день | Повний протокол + PAP-підхід 90–95% за 8–12 хв до виходу |
| Без залу | Загальна 2 хв → RDL з власною вагою 2×8 → good morning 1×10 → hip hinge з паузою |
| Поганий стан | Стандартний протокол → діагностика RPE на 65%: якщо ≥ 8 → знижуй або скорочуй об'єм |

**Чому так:**
- Становий підйом має вищу вимогу до кульшового і люмбопельвічного при збереженій нейтралі під навантаженням. Розгиначі спини активуються без стомлення — bird-dog або аналог, але не гіперекстензія до відмови.
- У форматі 5 хвилин: навантажений підхід починається з 40%, не нижче, бо при важкій тязі низький відсоток дає неправильний патерн зусилля.
- Змагальний день: PAP-підхід в тязі — повний, зі змагальним обладнанням (взуття, пояс, лямки якщо використовуються).

## Матриця: жим лежачи

| Умова | Протокол |
|-------|----------|
| Стандарт (20+ хв) | Загальна 3 хв → грудний відділ 2 хв → зовнішня ротація 2 хв → лопаткове навантаження 2 хв → навантажені підходи 30–90% |
| 10–15 хв | Загальна 2 хв → зовнішня ротація 1–2 хв → навантажені підходи від 40% |
| 5 хв | Загальна 90 с → 3 навантажені підходи (35%, 55%, 70%) |
| Змагальний день | Повний протокол + PAP-підхід 90–95% за 8–12 хв до виходу |
| Без залу | Загальна 2 хв → відтискання з паузою 2×8 → зовнішня ротація з гумкою → pike push-up |
| Поганий стан | Стандартний протокол → діагностика RPE на 65%: якщо ≥ 8 → знижуй робочу вагу |

**Чому так:**
- Жим лежачи — найбільш чутливий до стану плечового комплексу. Зовнішня ротація зберігається навіть при 10-хвилинному форматі, бо вартість пропущеної підготовки плеча вища за 2 хвилини часу.
- При 5 хвилинах — грудний відділ і лопаткова підготовка вирізаються. Це прийнятний компроміс, якщо немає попередніх обмежень.
- Детальний протокол → пост 1724.

## Матриця: підтягування / горизонтальна тяга

| Умова | Протокол |
|-------|----------|
| Стандарт (20+ хв) | Загальна 3 хв → лопаткові стабілізатори 2 хв → грип-активація 1 хв → dead hang 30 с → навантажені підходи або розмінні підтягування |
| 10–15 хв | Загальна 2 хв → dead hang 30 с → розмінні підходи від легкого |
| 5 хв | Загальна 90 с → 2 розмінні підходи (50–60% від максимального навантаження) |
| Змагальний день | Повний протокол; PAP рідко застосовується у підтягуванні — використовувати тільки при специфічній підготовці |
| Без залу | Загальна 2 хв → doorframe row або нижній кут для тяги → scapular push-up 2×10 |
| Поганий стан | Стандартний протокол → перший робочий підхід — не з максимальною вагою; скорочуй навантаження |

**Чому так:**
- Підтягування і горизонтальна тяга мають нижчий ризик «холодного старту» порівняно з жимом, але лопаткова стабільність критична. Dead hang в будь-якому протоколі — дешевий спосіб перевірити стан плеча.
- PAP при підтягуванні — нестандартне застосування; ефект менш передбачуваний і досліджений менше, ніж у жимах або присіданні.

## Матриця: жим стоячи (overhead press)

| Умова | Протокол |
|-------|----------|
| Стандарт (20+ хв) | Загальна 3 хв → грудний відділ 2 хв → зовнішня ротація 2 хв → lat tight і core-активація 1 хв → навантажені підходи 30–85% |
| 10–15 хв | Загальна 2 хв → зовнішня ротація 1–2 хв → навантажені підходи від 40% |
| 5 хв | Загальна 90 с → 3 навантажені підходи (35%, 55%, 70%) |
| Змагальний день | Повний протокол; PAP — за потребою, не стандарт для OHP |
| Без залу | Загальна 2 хв → pike push-up → зовнішня ротація з гумкою → wall slide |
| Поганий стан | Стандартний протокол → якщо RPE на 65% ≥ 8 → знижуй |

**Чому так:**
- OHP — найбільш технічно вимогливий жим через стабілізацію core і lat tight. При обмеженому часі зовнішня ротація і lat-активація мають вищий пріоритет, ніж грудний відділ (якщо немає специфічних обмежень мобільності грудного).
- Відрізняється від жиму лежачи: не arch, а neutral spine і стиснений живіт.

## Зведена пріоритизація при скороченні

При нестачі часу елементи вирізаються в такому порядку (першими вирізаються перші):

1. Профілактична мобілізація зон без поточних обмежень
2. Специфічна мобілізація (грудний відділ, гомілковостопний)
3. Активаційні вправи (CARs, bird-dog, лопаткові)
4. Зовнішня ротація — вирізається тільки при 5-хвилинному форматі
5. Навантажені підходи — не вирізаються ніколи
6. Загальна розминка — не вирізається ніколи

**Останні два пункти — нескорочувані.** Це принцип з постів 1711–1713: функція над об'ємом.

## Одне правило для всіх клітин матриці

Будь-який протокол у таблиці ліпший за відсутність протоколу. 5-хвилинна розминка — не «неповна». Вона закриває 80% пріоритетів. «Немає часу» означає «є 5 хвилин» — завжди.

Матриця не заміняє суджень. Вона позбавляє від необхідності думати про структуру, коли є час думати про підхід.
