### Block 901–950 — Practical athlete intelligence (proposed · v4.57.0)

Не наука і не техніка — а мислення атлета. Як приймати рішення у тренінгу без тренера, без досліджень під рукою і без ідеальних умов. Пости для атлета, який вже знає принципи і тепер вчиться застосовувати їх у реальному житті.

| Topic |
|---|

| Що робити, коли сесія йде не так, як заплановано |
| Різниця між «погана сесія» і «погана програма» |
| Читання тренувального журналу як діагностики, а не архіву |
