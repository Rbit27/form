# Sample Cutting Plan · 12 Weeks

> **⚠️ SAFETY GATE:** This plan активний тільки якщо ED-screening passed
> на онбордингу. Soft-mode users НЕ отримують його.

---

## Принципи (anti-toxic)

1. **Slow > fast.** 0.5-0.75% bodyweight loss per week max. > 1% = muscle loss.
2. **Protein high.** 2.0-2.4 g/kg lean. Це non-negotiable.
3. **Calorie floor: 1800 kcal/day для 85kg user.** Не нижче.
4. **No "cheat days" planned.** Структура з flexibility, не з guilt.
5. **Weekly weigh-in only.** Same time, same day.
6. **Strength expected to MAINTAIN, not grow.** Якщо decline > 5% — calories ↑ або pause.
7. **Hunger okay, suffering not okay.** Sub-jective rating; if < 3/5 для тиждня — re-evaluate.

**Critical:** Перехід phase → phase based on metrics, not calendar. If signals weak — stay.

## Monitoring (weekly review)

### Track these (in app)

| Metric | Target trend | Action if off |
|---|---|---|
| Weight | -0.5kg/week | Stay course |
| Strength (top sets) | Stable | Calories ↑ якщо declining 5%+ |
| HRV | Stable / slight ↓ ok | If declining 15%+ → ↑ calories |
| Sleep | Maintained | If < 6h average → ↑ calories |
| Mood/energy | 4-5 / 5 | If < 3 chronically → pause cut |
| Hunger | 3-4 / 5 | If 5/5 chronically → ↑ calories або add bulk fiber |

### Weekly weigh-in protocol

- Same day of week (suggested: Saturday morning)
- Same time
- After bathroom, before breakfast
- Underwear only
- Use 3-week rolling average, not raw weekly number
- Body changes lag actual trend by 1-2 weeks

## What to do at end of 12 weeks

### Option A: Take a break (recommended for most)
- Maintenance calories for 4-6 weeks
- Eat back to TDEE
- Train hard, recover, rebuild
- Then optionally another cut later

### Option B: Mini bulk
- +200 kcal over TDEE
- Build muscle for next cut

### Option C: Continue cutting (rarely advised)
- Only якщо still very high body fat
- Take diet break (2 weeks at maintenance) every 4-8 weeks of cutting

**Never plan continuous deficit > 16 weeks без breaks.**

## Signs to STOP this plan

Stop and re-evaluate (or contact clinical-safety) if:

- Weight dropping > 1.5% per week consistently
- Strength declining > 10%
- Sleep efficiency drops below 80%
- Mood persistently low (3+ weeks)
- Hunger constantly maxed out (suffering, not discomfort)
- Hair loss, brittle nails, dry skin
- Cold all the time
- Lost period (women)
- Obsessive food thoughts
- Avoidance of social eating
- Binge episodes

**Any of last 4 → talk to a doctor + therapist. Not a fitness app issue.**
